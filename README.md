# 🤖 AI-Powered LinkedIn Job Scraper

An AI-powered LinkedIn Job Scraper built with Python, Gradio, BeautifulSoup, and Flair NLP. This application scrapes LinkedIn job listings, extracts job details, identifies required skills using AI, and exports results to CSV.

## ✨ Features

- 🔍 Search LinkedIn jobs by keyword
- 🌍 Filter by city and country
- 💼 Filter by work type (Remote, Hybrid, On-site)
- 📈 Filter by experience level
- 📅 Filter by posting date
- 🤖 AI-powered skill extraction using Flair NLP
- 📄 Extract complete job descriptions
- 💾 Export results to CSV
- 🖥️ Interactive Gradio web interface

---

## 🛠️ Technologies Used

- Python
- Gradio
- BeautifulSoup4
- Requests
- Pandas
- Flair NLP
- PyTorch
- lxml

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/khashirmehmood09-ui/linkedin-job-scraper.git
cd linkedin-job-scraper
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
